## :rocket: Run

```
npm i
export REACT_APP_A3RT_TALK_API_KEY=$YOUR_A3RT_TALK_API_KEY
export REACT_APP_A3RT_IMAGE_SEARCH_API_KEY=$YOUR_A3RT_IMAGE_SEARCH_API_KEY
export REACT_APP_NOBY_API_KEY=$YOUR_NOBY_API_KEY

# Development
npm start
-> https://localhost:3000

# Production
npm run build
-> ./build
```
